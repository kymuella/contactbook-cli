# Contact Book CLI

A simple command-line Contact Book in Python

## Features

- Add contacts into a stored dictionary
- View contact list
- Search through contacts by inputting a name
- Remove contact from list
- Exit program

## How to run 
1. Make sure Python is installed
2. Run: `python contactbook.py`
3. Follow on-screen prompts

## What I learned
- Using dictionaries to store structured data (name + phone per contact)
- Debugged a tricky search bug where a "found" flag kept getting reset 
  to False on every non-matching contact instead of only being set once
- Learned the difference between code that runs once after a loop 
  finishes vs. code that accidentally runs on every loop iteration, 
  after fixing several indentation bugs
